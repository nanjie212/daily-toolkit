# 普通日常工具箱 — 产品白皮书

> 版本：v1.1.0 | 更新日期：2026-05-22  
> 核心理念：**99% 本地化运行 · 永久免费 · 无需注册 · 零数据上传**

---

## 一、网站概述

**普通日常工具箱**是一个纯前端、本地化的多功能工具集合网站。所有工具均在浏览器端运行，用户的数据不会离开本地设备。项目基于 React 18 + TypeScript + Vite 构建，部署于 Cloudflare Pages。

### 核心技术栈

| 层面 | 技术选型 |
|------|---------|
| 框架 | React 18 + TypeScript |
| 构建 | Vite 6 |
| 样式 | Tailwind CSS 3 |
| 状态管理 | Zustand |
| 路由 | React Router DOM (Hash模式) |
| 部署 | Cloudflare Pages (GitHub自动部署) |
| 云存储 | Supabase (限社区留言板，可选) |
| 本地存储 | localStorage |

## 四、产品哲学

### 4.1 设计理念

```
少即是多       → 功能丰富但界面简洁，不堆砌
隐私优先      → 数据不出本地，工具不上云
普惠工具      → 像瑞士军刀一样，日常随手可用
社区共建      → 工具市场由用户贡献
```

### 4.2 不做的事（铁律）

- ❌ 不上传用户数据（所有计算在浏览器内完成）
- ❌ 不收集用户行为（零埋点、零分析脚本）
- ❌ 不加广告（网站运行成本为零，没有加广告的任何理由）
- ❌ 不加付费墙/会员体系
- ❌ 不强制联网（字体/图标/PWA 全部本地自托管，CDN 仅用于 OCR 等需联网的功能）
